# Automate Hedera Hashgraph with n8n: Introducing Our New Community Node

We're excited to announce the release of our latest n8n community node: n8n-nodes-hedera-hashgraph! At Velocity BPA, we're constantly working to bridge the gap between powerful blockchain technologies and accessible automation tools.

## The Challenge

Hedera Hashgraph has emerged as a leading enterprise-grade distributed ledger technology, offering fast, fair, and secure transactions. However, integrating Hedera into existing workflows has traditionally required significant development resources. Teams needed custom code to handle HBAR transfers, manage HTS tokens, work with NFTs, or leverage the Hedera Consensus Service for messaging.

For businesses using n8n as their automation platform, this meant either building custom solutions from scratch or maintaining separate systems—neither of which is ideal for rapid deployment and iteration.

## Introducing n8n-nodes-hedera-hashgraph

Our new community node brings Hedera Hashgraph capabilities directly into your n8n workflows with a user-friendly, no-code interface. Whether you're building DeFi applications, managing tokenized assets, or creating decentralized communication systems, you can now automate it all within n8n.

## Key Features

**HBAR Transfers**: Send and receive HBAR cryptocurrency with simple workflow nodes. Perfect for automated payment processing, treasury management, or reward systems.

**HTS Token Operations**: Create and manage Hedera Token Service (HTS) tokens directly from your workflows. Mint, burn, transfer, and query token balances without writing a single line of code.

**NFT Management**: Handle NFT creation, transfers, and queries. Ideal for businesses building digital collectibles, proof-of-authenticity systems, or tokenized asset platforms.

**HCS Messaging**: Leverage the Hedera Consensus Service for immutable, timestamped messaging. Great for audit trails, supply chain tracking, or decentralized application communication.

## Getting Started

Installation is straightforward. In your n8n instance, simply run:


npm install n8n-nodes-hedera-hashgraph


After restarting n8n, you'll find the Hedera Hashgraph node available in your node palette, ready to integrate into your workflows.
